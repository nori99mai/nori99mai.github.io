02 COMMANDS
COMMAND DESCRIPTION
/splapena run VCにいる全員にペナルティをランダムで割り当てて発表
/splapena list 登録中のペナルティ一覧を表示
/splapena add ペナルティを追加（サーバー管理者のみ）
/splapena remove ペナルティを削除（サーバー管理者のみ）
/splapena reset ペナルティをデフォルト33件にリセット（サーバー管理者のみ）
/splapena edit 登録ペナルティを1画面で一括編集（サーバー管理者のみ）

BOT導入後、VCのチャットで /splapena と入力すると候補が表示されます。選ぶだけでOK、スマホでも簡単です。
ペナルティはもともと33個入っており、好きなものを追加・削除できます（サーバーごとに管理）。

04 UPDATE
UPDATE — 2026.06.23

不具合の修正・新機能の追加・抽選バランスの改善を行いました。

1. runの前にaddをすると、デフォルトペナが登録されないエラーを修正

※6/23 23:00頃に修正を反映しました。

このbotには33種類のデフォルトペナが、5カテゴリに分かれて登録されています。

ところが、デフォルトを使う（run）より先にadd（ペナ追加）をすると、BOTちゃんが、
「あ！鯖主さんが追加してくれたペナルティちゃんを優先しなきゃ！」
と判断して、デフォルトペナを取り込まないエラーがありました。
（その結果、その鯖にはデフォルトが1つも入っていない状態に…！）

現在はBOTちゃんに、
「addのときも、デフォルトペナも鯖主さんペナも、ちゃんと両方活かしてね」
と教育済みです。

ただし、6/16〜6/24にこの操作をしてしまった鯖は、すでにデフォルトが入っていない場合があります。

その場合は、下のedit機能で復元できます ↓

2.（新機能）edit機能を実装

登録されているペナルティを一覧で編集できる機能を付けました。
/splapena edit を実行して、下のデフォルトペナ一覧を貼り付けて登録することも可能です。
ちなみにここでペナルティの一括編集もできちゃいます！

3. 登録上限を最大150件に拡張

たくさんカスタムしても余裕の容量に。一覧が長くなっても、自動で複数メッセージに分けて表示します。

4. 出現率の調整について

「好きなペナルティを誰かに押し付けられる」これはスペシャルなjokerカードとして用意したものですが、実際に試してみたら結構な確率で出現しておりました笑

その他にもいろいろなペナルティを登録していくと、母数が増えていくため全く出てこなくなるペナルティが出てきてしまう問題がありました。

この2つの課題解決のために、下記の修正をしています。
・最初に、各ペナルティのカテゴリをランダムで選ぶ。
・次に、選ばれたカテゴリ内でランダムでペナルティを選択。

こうすることで色々なペナルティの出現率を調整、またこの調整の上でjokerの出現率のみ少し下げております。
