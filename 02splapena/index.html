DISCORD BOT

by nori99mai

Splatoon3 プライベートマッチを
盛り上げるDiscord BOTです。
VCにいる全員へペナルティを
ランダム割り当て。
ちょっとした遊びのルール決めにどうぞ！

01 HOW TO INVITE
FOR ADMIN

サーバー管理者の方は下のボタンからDiscordサーバーに招待できます。
インストール不要・無料です。

INVITE TO DISCORD →
FOR MEMBERS

このBotはサーバー管理者のみが導入できます。
使いたい場合は、このページのURLをサーバーの管理人さんへお伝えください。

02 COMMANDS
COMMAND DESCRIPTION
/splapena run VCにいる全員にペナルティをランダムで割り当てて発表
/splapena list 登録中のペナルティ一覧を表示
/splapena add ペナルティを追加（サーバー管理者のみ）
/splapena remove ペナルティを削除（サーバー管理者のみ）

BOT導入後、VCのチャットで /splapena と入力すると候補が表示されます。選ぶだけでOK、スマホでも簡単です。
ペナルティはもともと19個入っており、好きなものを追加・削除できます（サーバーごとに管理）。
