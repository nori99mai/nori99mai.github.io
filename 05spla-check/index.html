これは何？

初心者が自分のリプレイ（メモリープレイヤー）を見返して、「やらかし」をボタンでポチって記録 → 弱点傾向を集計するアプリの、チェック項目リストです。多数の添削動画から「XP2000到達に必要な“できて当然の基礎”」を抜き出し、初心者が一目で『あ、これやった』と分かる最小単位に分解しました。すでに身内1名とブラッシュアップ済みで、今回はレート2700クラスの視点での最終チェックをお願いしたいものです。

見ていただきたい2点

  • 的外れでないか … XP2000到達の基礎としてズレていないか
  • 抜けはないか … 定番のやらかしで抜けているものはないか
あえて多め（28個）に出しています。「足す」より 「消す・直す・統合する」 視点で見てもらえると助かります。
各項目に ○=残す / △=直したい / ✕=消す ＋一言コメント（直し案・統合案など）をいただけると最高です。

※各項目の 「言及◯」 ＝ 参考にした添削動画群のうち、その項目に触れていた本数の目安です（多いほど“定番のやらかし”）。重要度の参考値で、確定ではありません。「追加」チップ＝今回の身内レビューで新しく加えた項目（動画の集計対象外）。
