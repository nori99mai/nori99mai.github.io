# assets ─ 木の絵を置く場所

ここに `stage1.png` 〜 `stage10.png` を置くと、アプリが**自動で仮の絵から差し替え**ます。
（1枚も無ければ仮のSVGの木が表示されます。プログラムの修正は不要。）

## ルール
- ファイル名：`stage1.png`〜`stage10.png`（連番・小文字）
- **stage1〜9＝お子さんの手書き**（テンプレートに描く → スキャン/撮影）
- **stage10＝AI絵**（実在しうる大樹。`../../sozai/prompts.md` 参照）
- 背景は**透過PNG**推奨（白背景でも可）
- **足元を画像の下端の同じ高さ**・**幹を中央**・**正方形に近い比率**で揃える
  → お絵描きテンプレート `../../sozai/01_ki/template.html` を使うと自然に揃う
- 表示が重い場合は**長辺〜1024pxに縮小**（spec ⑦-3 パフォーマンス）

## 段階の目安（現実サイズ）
1 芽 / 2 双葉 / 3-4 若木 / 5-6 中木 / 7-8 大きな木 / 9 立派な大木 / 10 実在級の巨木（屋久杉・セコイア）
